# 🎬 Anime Discovery Web App

A modern full-stack web application to explore anime using a clean UI and real-time data. Built with **Spring Boot**, **Maven**, and **Docker**, this project demonstrates containerized deployment and API integration.

---

## 🚀 Features

- 🎯 Browse anime by categories (Action, Romance, Comedy, etc.)
- 🔍 Search anime using real-time API
- 📄 View detailed anime info (rating, synopsis, poster)
- 🎨 Modern UI with animations and hover effects
- 🌙 Dark / Light mode toggle
- ⚡ Smooth transitions and loading animations
- 📱 Responsive design

---

## 🛠️ Tech Stack

- **Backend:** Java, Spring Boot  
- **Frontend:** HTML, CSS, Thymeleaf  
- **Build Tool:** Maven  
- **API:** Jikan API (MyAnimeList)  
- **Containerization:** Docker & Docker Compose  
- **IDE:** VS Code
